## 🎯 项目概述

### 核心特性

- **🎮 数据规模**: 13万+ 条游戏数据记录 (怪物、武器、防具、技能、物品等)
- **🏗️ 技术架构**: Next.js + NestJS + PostgreSQL + Prisma
- **📦 项目管理**: Turborepo Monorepo
- **🚀 部署方案**: 宝塔面板 + PM2 + Nginx (无 Docker)
- **🌏 多语言支持**: 14种语言支持 (中文、英文、日文等)

### 技术特点

- 🔧 **宝塔友好** - 专为宝塔面板环境设计
- 📦 **无 Docker** - 简化部署流程，降低维护成本
- 🤖 **自动化脚本** - 一键部署和数据备份
- 📊 **性能监控** - PM2 进程管理和状态监控


## 🚀 快速开始

### 1. 环境要求

- Node.js >= 18
- PostgreSQL >= 13
- Git

### 2. 项目安装

```bash
git clone https://github.com/zcanic/mhw2wiki.git
cd mhw2wiki
npm install
```

### 3. 数据库配置

```bash
# 复制环境变量模板
cp packages/database/.env.example packages/database/.env

# 编辑数据库连接
nano packages/database/.env
```

### 4. 数据库初始化

```bash
cd packages/database

# 生成 Prisma 客户端
npm run db:generate

# 同步数据库结构  
npm run db:push

# 导入游戏数据 (约2-5分钟)
npm run db:seed
```

### 5. 开发服务器

```bash
# 启动开发服务器
npm run dev

# 或者分别启动前后端
npm run dev --workspace=apps/frontend
npm run dev --workspace=apps/backend
```
